Spring Mvc Url和参数名称忽略大小写

在开发过程中Spring Mvc 默认 Url和参数名称都是区分大小写的

例如：
  www.a.com/user/getUserInfo?userId=1
  www.a.com/user/getuserInfo?userId=1
  www.a.com/user/getUserInfo?userid=1
  www.a.com/user/getuserinfo?userid=1

这些都认为不同的地址和参数，在实际中用户根本不区分这些，所以我们要忽略大小写
